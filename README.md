# sasohan-chat

Chatting API server of [sasohan](https://github.com/insanesclub/sasohan)

## Supported functions

- [x] connect
- [x] disconnect
- [x] new chat
- [x] delete chat
- [x] leave chat

**Install**

```bash
$ GO111MODULE=on go get github.com/insanesclub/sasohan-chat
```

**Build**

```bash
$ make
```
